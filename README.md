# vim
Vim configuration for personal use.

## how to use this repository
1. clone this repository into ~/vim
2. link your local ~/.vimrc file to ./vimrc
``` shell
ln -s ~/.vimrc ~/vim/vimrc
```
3. link your local ~/.vim directory to ./vim/vim
``` shell
ln -s ~/.vim ~/vim/vim
```
